📖 User Guide
📚 Getting Started

Welcome to NUR265 CAPSTONE Supporting Resources! This application provides essential clinical study materials to support your nursing capstone journey.

📂 Browsing Documents

Use the left sidebar to navigate your document library. Click on folder names to expand or collapse them. Click on any document to view it in the main panel.

📤 Uploading PDFs

Switch to the "Upload" tab to add your own PDF files. Drag and drop a file onto the upload area, or click to browse your device. Uploaded files are stored temporarily in your browser session.

🔍 Viewing & Navigation

Use the navigation bar below each PDF to move between pages, zoom in/out, or close the document. You can also use keyboard shortcuts for faster navigation.

🔒 Protected Documents

Some documents in the Private folder require a password to access. Enter the correct password when prompted to unlock and view these files.

⌨️ Keyboard Shortcuts
Page Navigation
  • ← ↑ Previous page
  • → ↓ Next page
Zoom Controls
  • + = Zoom in
  • − Zoom out
Tips

• Use the Settings menu to set a default zoom level that applies to all documents.

• On mobile devices, use the navigation buttons below the PDF for easy control.
